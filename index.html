A Google Chrome extension that replaces new tabs with a text editor.
Built with quill.js.

Features

  • standard text editor

  • code-fencing and inline math

  • Markdown and Emoji shortcuts

Installation

  1. Clone this repo.

  2. Go to chrome://extensions/ via the address bar.

  3. Enable developer mode by clicking the toggle switch on the top right.

  4. Click load unpacked and select the internal markup folder.
